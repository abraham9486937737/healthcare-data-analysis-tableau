This project is based on a healthcare dataset containing 6,000 patient records. I analyzed the data to understand patient demographics, diagnosis trends, treatment costs, and department-wise performance. All visualizations were created in Tableau using interactive dashboards and story features.

Objective

To explore patient data and identify useful insights that can help in hospital planning and decision-making. The focus is on demographics, diagnosis categories, admission trends, and treatment-related patterns.

Tools Used

Tableau

CSV dataset (6,000 records)

Mackroo website for data preparation

Key Insights

Total patients: 6,000

Average age: 49.81 years

Male: 2,699, Female: 2,692

Top diagnoses include Diabetes, Hypertension, CKD, Lung Cancer, Heart Disease, and Asthma

General Medicine has the highest billing

Pediatrics department has the highest number of patients

Average treatment cost is $49,959

Patient admissions are evenly spread across the year

Conclusion

This Tableau dashboard helps visualize patterns in patient load, diagnosis frequency, and treatment costs. These insights can support better hospital management, resource planning, and decision-making. The dataset is AI-generated and used only for learning and practice.
